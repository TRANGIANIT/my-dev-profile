# LeetCode Skill

## Study Pattern

1. Understand problem
2. Write examples
3. Find brute force solution
4. Optimize
5. Analyze time complexity
6. Analyze space complexity
7. Review after 1 day, 1 week

## Important Patterns

- Two pointers
- Sliding window
- Hash map
- Stack
- Binary search
- DFS / BFS
- Dynamic programming
