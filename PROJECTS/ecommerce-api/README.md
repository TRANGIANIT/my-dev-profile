# Ecommerce API

## Overview

A sample Spring Boot backend project for ecommerce API.

## Features

- Product management
- User authentication
- Order management
- Validation
- Global exception handling

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Docker

## Architecture

```txt
Controller -> Service -> Repository -> Database
```

## Run Project

```bash
mvn clean test
mvn spring-boot:run
```

## Docker

```bash
docker compose up --build
```
